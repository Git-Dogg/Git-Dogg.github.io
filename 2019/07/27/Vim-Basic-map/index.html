VIM基础知识

1. vim快速移动

  • vim上下左右移动 hjkl
    • h (左) 、j (下) 、k (上) 、l (右)
  • vim 单词级移动
    • w/W 移动到下一个 word/WORD 开头；e/E 移动到下一个 word/WORD 尾
    • b/B 回到上一个 word/WORD 开头，可以理解为 backword
    • word 是指以非空白符分割的单词，WORD以空白符分割的单词
  • vim 行间搜索移动
    • 使用 f{char} 可以移动到char 字符上，t 移动到char 的前一个字符
    • 可以使用分号 (;) 或者逗号 (,) 搜索该行下一个或者上一个字符
    • F 表示前向搜索
  • vim 水平移动
    • 0 移动到行首第一个字符，^ 移动到第一个非空白字符
    • $ 移动到行尾，g_ 移动到行尾非空白字符
  • vim 垂直移动
    • 在句子 (sentence) 和段落 (paragraph) 间移动
    • 使用括号 () 在句子间移动，使用花括号{}在段落间移动
    • easy-motion 插件
  • vim 翻页移动
    • gg/G 移动到文件开头/结尾，ctrl + o 快速返回
    • H/M/L 跳转到当前屏幕的开头 (Head) 、中间 (Middle) 、结尾 (Lower)
    • Ctrl + u、Ctrl + f 上下翻页 (upward/forward) ，zz 将当前光标行放置到屏幕中间

2. vim快速增删改查

  • vim快速删除
    • vim在normal模式下使用x 快速删除一个字符
    • 使用d (delete) 配合文本对象快速删除一个单词daw (d around word)
      • dw 默认是daw 还有一种diw 不删除空格
      • dd 删除光标所在行
      • dt{char} delelte till
      • d$ 、d0 删除到 结尾、开头
    • d 和 x 都可以搭配数字来执行多次
    • d 配合 v (选择模式) V (行选模式) 删除
  • vim快速修改
    • 常用有三个，r (replace) ，c (change) ，s (substitute)
    • normal模式下使用 r 可以替换一个字符；s替换并进入插入模式
    • r 和 s 都可以搭配数字来使用
    • normal模式下使用R 将进入不间断输入替换模式；S将整行替换并进入插入模式
    • 使用c配合文本对象，可以进行快速修改
      • cw (change word) 修改一个word
      • ct{char} 修改到指定字符
  • vim增加字符
    • 进入插入模式编辑文本
    • 使用 a/i/o 进入插入模式
    • 使用 A/I/O 进入插入模式
  • vim快速查询
    • 使用 / 或者 ? 进行前向或者反向搜索
    • 使用 n/N 跳转到下一个或者上一个匹配
    • 使用 * 或者 # 进行当前单词的前向和后向匹配

3. vim搜索替换

  • vim替换命令
    • substitute命令允许我们查找替换文本，并支持正则
    • :[range]s[ubstitute]/{pattern}/{string}/[flags]
      • range 表示范围 比如 :10,20 表示10-20行，%表示全部
      • pattern 是要替换的模式，string是替换后文本
      • pattern 例如 精确匹配this %s/\<this>//n
    • flags 几个常用的标志
      • g (lobal) 表示全局范围内执行
      • c (onfirm) 表示确认，可以确认或者拒绝修改
      • n (umber) 报告匹配到的次数而不替换，可以用来查询匹配次数

4. vim复制粘贴

  • vim normal 模式复制粘贴
    • normal模式下复制粘贴分别用y (ank) 和p (ut) ，剪切d和p
    • 可以用v (isual) 命令选中所要复制的地方，然后使用p粘贴
    • 配合文本对象：比如使用yiw复制一个单词，yy 复制一行，然后p粘贴
  • vim insert 模式复制粘贴
    • insert 模式下复制粘贴根据使用系统、终端的不同，有以下几种方式
    • ms-windows Ctrl-c Ctrl-v
    • mac-os Cmd-c Cmd-v
    • xshell Ctrl-insert Shift-insert
    • 如果在 vimrc 中设置了autoindent，极有可能造成粘贴格式错乱，可以临时做如下处理
    • 在粘贴之前进行 :set paste，粘贴完成之后进行 :set nopaste

PS: 建议一边操作一边加深理解vim哲学，可以在normal模式下，使用 :help {命令} 来查看命令的详细解释
